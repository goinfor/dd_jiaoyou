# 在线交友微信小程序

### 介绍
一款交友小程序
### 功能列表
+ 展示所有用户列表
+ 查看用户信息
+ 拨打电话
+ 复制微信号
+ 修改用户信息
+ 展示附近用户
+ 添加好友
+ 发送添加好友信息
+ 展示好友列表
